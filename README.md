# vue-template

This template should help get you started developing with Vue 3 in Vite.

* 1、change package.json name, delete node_modules, run npm install
* 2、Upload favicon.ico to public folder
* 3、Update vite.config.ts Sitemap hostname
* 4、[option] add language support: ar、bn、de、en、es、fr、hi、id、it、ja、ko、pt、ru、ur、vi、zh-CN、zh-TW、nl、sv、no、da、fi、pl、cs、sk、hu、ro、bg、el、tr、he、fa、th、ms、ta、te、ml、kn、mr、gu、pa、sw、af、zu、xh、am、my、km、lo、si、ne、uk、sr、hr、sl、lt、lv、et、is、ga、cy、eu、gl、ca、eo、ka、uz-Latn
